How It Works

Context-Aware Intelligence

Every command automatically analyzes your project environment by running shell commands to gather context about code files, dependencies, and recent changes.

Official Google AI CLI Compliance

Commands use native `{{args}}` parsing for dynamic arguments and `!{command}` for shell integration, following the official specifications for robust and reliable execution.

description = "Brief description for /help menu"

prompt = """
You are a [role] specializing in [domain].

Task: {{args}}

## CONTEXT ANALYSIS
Project files: !{find . -name "*.ext" | head -5}
Dependencies: !{cat package.json 2>/dev/null}

## CAPABILITIES
- Capability 1 with clear parameters
- Capability 2 with usage examples

[Implementation guidance]
"""

Quick Start

Native Discovery (Recommended)

Copy commands to your Google AI CLI directory for automatic discovery and use them directly inside the TUI.

# Global installation
cp commands/engineering/security-analyst.toml ~/.gemini/commands/

# Inside Gemini TUI
/security-analyst deps scan

On-Demand Loading

Load any command temporarily for a single execution without needing to install it.

# Terminal command
gemini --include commands/eng/ai-engineer.toml "/ai-engineer llm openai"
