How It Works

Context-Aware Intelligence

Each command automatically analyzes your project environment, executing shell commands to gather context about code files, dependencies, and recent changes.

Built Specifically for the Google AI CLI

These commands were designed from the ground up to utilize the native `{{args}}` syntax for dynamic arguments and `!{command}` for shell integration, following official specifications for robust and reliable execution.

Quick Start

Native Discovery (Recommended)

Copy the TOML files to your Google AI CLI commands directory. The CLI will automatically discover them, allowing you to use them directly inside the TUI (Terminal User Interface).

# Global installation
cp commands/engineering/security-analyst.toml ~/.gemini/commands/

# Inside the Gemini TUI
/security-analyst deps scan

On-Demand Loading

Temporarily load any command for a single execution, with no need to install it.

# Command in the terminal
gemini --include commands/eng/ai-engineer.toml "/ai-engineer llm openai"
